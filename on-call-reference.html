Quick Procedures

DOWNTIME PROCEDURE

  1. Contact IT Support: 301-456-6096
  2. Initiate manual backup procedures
  3. Log all results manually
  4. Notify all affected departments
  5. Document in incident report

Critical Value Reporting

  1. Call provider IMMEDIATELY
  2. Document: Date, Time, Who received
  3. Repeat back confirmation
  4. Enter in LIS within 15 minutes
  5. If no response in 10 min ‚Üí Escalate to charge nurse

Equipment Failure

  1. Check error code and document
  2. Call appropriate vendor support
  3. Switch to backup instrument if available
  4. Notify supervisor/manager
  5. Update instrument log
