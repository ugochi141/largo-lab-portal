üìã Equipment Downtime Procedures

Roche Pure Analyzer Downtime

  1. Contact Roche Support immediately: 1-800-428-2336
  2. Provide account number: 55042629
  3. If MOB Pure 1 is down, redirect STAT tests to AUC Pure 2
  4. If both analyzers down, implement manual backup procedures
  5. Notify all phlebotomy stations of extended TAT
  6. Document downtime in maintenance log
  7. For critical tests, contact reference lab

Sysmex XN-Series Downtime

  1. Contact Sysmex Support: 1-888-879-7639
  2. Provide account number: MOB (2004001963) or AUC (2004017492)
  3. If one XN-2000 is down in MOB, use the other unit
  4. If both MOB units down, redirect to AUC XN-1000
  5. For manual differentials, use backup microscopes
  6. Critical values must be reviewed immediately
  7. Implement manual counting procedures if necessary

Stago Compact Max Downtime

  1. Contact Stago Support: 800-725-0607
  2. Provide account: MOB (1397598) or AUC (1224867)
  3. If MOB Stago down, redirect to AUC Stago
  4. For critical INR results, use point-of-care testing
  5. Notify anticoagulation clinic of delays
  6. Store specimens at appropriate temperature for later testing
  7. Document all delayed tests in log

Urinalysis Equipment Downtime

  1. For UF-5000: Contact Sysmex at 1-888-879-7639
  2. For Clinitek Novus: Contact Siemens at 1-888-879-7639
  3. Use manual microscopy for urgent urines
  4. Dipstick testing can continue manually
  5. Document all manual results properly
  6. Store specimens for later automated testing

GeneXpert Downtime

  1. Contact Cepheid: 1-888-838-3222 Option 2
  2. Provide serial numbers: 844498 / 210015992
  3. For urgent COVID/Flu testing, contact reference lab
  4. Store specimens at appropriate temperature
  5. Notify infection control if critical tests delayed
  6. Document all specimens for batch testing when operational
