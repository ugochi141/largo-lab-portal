MANUAL DIPSTICK ANALYSIS

Indications for Manual Testing

Perform manual dipstick analysis only when:

  • CLINITEK Novus is out of service
  • Emergency backup testing is required
  • Confirmation of automated results is needed
  • Special circumstances as directed by supervisor

Materials Required

  • Multistix 10 SG or equivalent reagent strips
  • Fresh urine specimen (well-mixed)
  • Timer or stopwatch
  • Color comparison chart (from reagent strip bottle)
  • Clean, dry container if transfer is needed
  • Absorbent paper

Pre-Analytical Considerations

IMPORTANT: Check reagent strip expiration date before use. Do not use expired strips.
  • Store reagent strips in original container at room temperature (15-30°C)
  • Keep container tightly closed when not in use
  • Do not remove desiccant from bottle
  • Do not touch test areas with fingers
  • Specimen should be at room temperature before testing

Procedure

  1. Specimen Preparation
    • Mix specimen gently but thoroughly by inverting container 8-10 times
    • Do not centrifuge specimen before chemical testing
    • Test within 2 hours of collection if kept at room temperature
  2. Remove Reagent Strip
    • Remove one strip from container
    • Immediately replace cap tightly
    • Do not remove strip until ready to test
  3. Immerse Strip
    • Completely immerse all reagent areas in fresh, well-mixed urine
    • Remove immediately - do not leave strip in urine
  4. Remove Excess Urine
    • While removing, run edge of strip against rim of container
    • Turn strip on its side and tap on absorbent paper
    • This prevents reagents from running together
  5. Begin Timing
    • Start timing immediately after removing excess urine
    • Hold strip horizontal during entire read time
    • Do not allow reagent areas to touch anything
  6. Read Results
    • Compare each test pad to corresponding color blocks on bottle label
    • Read at exact times specified for each test
    • Read in good lighting conditions
    • Record results immediately

Reading Times and Interpretation

Test Parameter Reading Time Normal Range Clinical Significance
Glucose 30 seconds Negative Screens for diabetes mellitus, renal glycosuria
Bilirubin 30 seconds Negative Liver disease, biliary obstruction
Ketone 40 seconds Negative Diabetic ketoacidosis, starvation, vomiting
Specific Gravity 45 seconds 1.003-1.030 Kidney concentrating ability, hydration status
Blood 60 seconds Negative Hematuria, hemoglobinuria, myoglobinuria
pH 60 seconds 4.5-8.0 Acid-base balance, UTI, renal stones
Protein 60 seconds Negative-Trace Renal disease, pre-eclampsia, multiple myeloma

Leukocytes 120 seconds Negative Urinary tract infection, inflammation

Important Notes on Reading

  • Timing is Critical: Read each parameter at specified time. Color development continues after specified time
  • Lighting: Use consistent lighting. Fluorescent lighting may affect color interpretation
  • Color Matching: Match colors carefully. If between two color blocks, report higher value
  • Documentation: Record results immediately to avoid errors

Interferences and Limitations

Glucose

  • False Negative: High specific gravity, elevated ketones, ascorbic acid
  • False Positive: Peroxide, oxidizing detergents

Bilirubin

  • False Negative: Ascorbic acid, light exposure
  • False Positive: Drug metabolites (phenazopyridine)

Ketone

  • False Positive: L-dopa metabolites, phenylketones
  • Note: Detects acetoacetic acid, not β-hydroxybutyrate

Blood

  • False Positive: Oxidizing contaminants, bacterial peroxidases
  • False Negative: Ascorbic acid, high nitrite
  • Note: Cannot distinguish intact RBCs from hemoglobin/myoglobin

pH

  • No known interferences
  • Run-over from adjacent protein pad may affect reading

Protein

  • False Positive: Highly alkaline urine (pH >8), quaternary ammonium compounds
  • False Negative: Dilute urine, non-albumin proteins
  • Note: More sensitive to albumin than other proteins

Nitrite

  • False Negative: Ascorbic acid, short bladder retention time, lack of dietary nitrate
  • False Positive: Medication causing red color
  • Note: Requires bacteria that reduce nitrate (E. coli, etc.)

Leukocytes

  • False Positive: Strong oxidizing agents, formalin
  • False Negative: High glucose, high protein, ascorbic acid

Result Reporting Format

Report results using the following standardized format:

  • Negative results: Report as "Negative"
  • Trace results: Report as "Trace"
  • Positive results: Report actual value from color chart
  • Use units specified on reagent bottle (mg/dL, etc.)

Confirmatory Testing

Perform confirmatory testing when:

  • Results don't correlate with clinical picture
  • Borderline or unexpected positive results
  • Interference is suspected

Confirmatory Tests Available:

  • Protein: Sulfosalicylic acid (SSA) precipitation
  • Bilirubin: Ictotest tablet method
  • Ketones: Acetest tablet method
  • Reducing substances: Clinitest tablet method

Quality Assurance for Reagent Strips

Daily Quality Control

  1. Control Testing
    • Test both positive and negative controls each day of use
    • Test new lot numbers before placing in use
    • Document all control results in QC log
    • Controls must be within acceptable ranges before patient testing
  2. Visual Inspection
    • Check for discoloration of pads
    • Ensure pads are firmly attached
    • Verify expiration date
    • Note any unusual odor upon opening container

Storage Requirements

  • Store at room temperature (15-30°C)
  • Do not refrigerate or freeze
  • Keep away from moisture, direct light, and heat
  • Do not use past expiration date
  • Write open date on container; discard after 6 months

Lot-to-Lot Verification

  1. Compare new lot with current lot using patient samples
  2. Test at least 5 samples covering range of results
  3. Results should agree within one color block
  4. Document comparison results

Troubleshooting Guide

Problem Possible Cause Action
All pads positive Contamination, expired strips Check expiration, test new bottle
No color development Strips exposed to moisture Replace with new bottle
QC out of range Expired controls, improper storage Check control dates, repeat with new
Inconsistent results Timing errors, poor technique Review procedure, retrain if needed

Documentation Requirements

  • Record lot number and expiration date
  • Document daily QC results
  • Note any corrective actions taken
  • Maintain temperature logs for storage area
  • Keep records for minimum of 2 years

Competency Assessment

Personnel performing manual dipstick analysis must:

  • Demonstrate proper technique initially and annually
  • Successfully complete color discrimination test
  • Show proficiency in timing and reading results
  • Understand limitations and interferences
  • Know when to perform confirmatory testing
