1. PURPOSE AND SCOPE

This Standard Operating Procedure (SOP) provides comprehensive guidelines for the performance of Human Chorionic Gonadotropin (HCG) testing in serum and urine specimens. This procedure is used for pregnancy confirmation, monitoring pregnancy progression, detecting ectopic pregnancy, and monitoring treatment of HCG-producing tumors.

Scope: This SOP applies to all laboratory personnel performing HCG testing using automated immunoassay platforms and point-of-care testing devices within Kaiser Permanente laboratory facilities.

4. SAFETY PRECAUTIONS AND HAZARD COMMUNICATION

Biological Safety:

  • Treat all specimens as potentially infectious
  • Use standard precautions for blood and body fluids
  • Wear appropriate PPE: gloves, lab coat, safety glasses
  • Work in biological safety cabinet when required
  • Proper disposal of biohazardous waste

Chemical Safety:

  • Review SDS for all reagents before use
  • Store reagents according to manufacturer specifications
  • Use fume hood for volatile substances
  • Emergency procedures readily available

7. SPECIMEN COLLECTION AND HANDLING

Acceptable Specimens:

  • Serum: Red top or gold top (SST) tubes
  • Plasma: Lithium heparin or EDTA
  • Urine: Random or first morning void (preferred)

Collection Requirements:

  • Proper patient identification
  • Appropriate tube selection
  • Adequate sample volume (minimum 0.5 mL serum)
  • Room temperature transport acceptable
  • Refrigerate if testing delayed >4 hours

Rejection Criteria:

  • Unlabeled or mislabeled specimens
  • Hemolyzed samples (severe)
  • Insufficient volume
  • Improper storage conditions
  • Expired specimens (>7 days)

9. QUALITY CONTROL

QC Requirements:

QC Level Frequency Acceptable Range Action if Out of Range
Level 1 (Negative) Each run <5 mIU/mL Repeat QC and investigate
Level 2 (Positive) Each run ±2 SD from mean Repeat QC and recalibrate
Level 3 (High) Each run ±3 SD from mean Check linearity

Westgard Rules Application:

  • 1:2s - Warning rule
  • 1:3s - Reject run
  • 2:2s - Reject run
  • R:4s - Reject run
  • 4:1s - Reject run
  • 10:x - Reject run

10. REFERENCE RANGES AND CRITICAL VALUES

Population Reference Range Units Clinical Significance
Non-pregnant females <5 mIU/mL Negative for pregnancy
Males <2 mIU/mL Normal level
Pregnant (varies by week) 25-288,000 mIU/mL Gestational age dependent

Critical Values:

  • Unexpectedly high HCG in males: >10 mIU/mL (possible tumor)
  • Low HCG in known pregnancy: May indicate ectopic pregnancy or miscarriage
  • Notification: Contact ordering physician within 1 hour

11. RESULT INTERPRETATION AND REPORTING

Interpretation Guidelines:

  • <5 mIU/mL: Negative for pregnancy
  • 5-25 mIU/mL: Equivocal, repeat in 48-72 hours
  • >25 mIU/mL: Positive for pregnancy
  • Serial monitoring: Should double every 48-72 hours in normal pregnancy

Reporting Format:

  • Quantitative results in mIU/mL
  • Qualitative interpretation when appropriate
  • Reference ranges included
  • Delta checks for serial results
